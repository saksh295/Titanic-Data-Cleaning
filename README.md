# Titanic Data Cleaning and Preprocessing

This repository contains a Python script performing data cleaning and preprocessing on the Titanic dataset for Elevate Labs.

## Steps Performed
1. Imported and explored dataset  
2. Handled missing values using mean/median/mode  
3. Encoded categorical variables  
4. Standardized numerical columns  
5. Visualized and removed outliers using IQR  
6. Saved cleaned dataset  

## Tools Used
- Python  
- Pandas, NumPy, Seaborn, Matplotlib  
- Scikit-learn (LabelEncoder, StandardScaler)

## Output
- Cleaned dataset: `titanic_cleaned.csv`
